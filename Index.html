1. Crie conta grátis em emailjs.com
2. Crie um Email Service e copie o Service ID
3. Crie um Email Template com variáveis {{to_email}}, {{plan_name}}, {{days_left}}, {{target}}, {{current}}, {{unit}} e copie o Template ID
4. Copie sua Public Key (Account > General)
5. Cole os dados abaixo e salve

Backup e Restauração
Sincronize os dados entre computador e celular
Como sincronizar com o celular:
1. No computador → clique em Exportar backup
2. Envie o arquivo .json para si mesmo (WhatsApp, e-mail, Drive)
3. No celular → abra o app → clique em Importar backup
4. Selecione o arquivo recebido — pronto! ✅
